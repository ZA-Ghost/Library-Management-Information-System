Welcome to the Library Management Information System!

Dashboard

  • Add Book
    • Leads to form
  • View Book
    • Leads to brief table, brief table leads to detailed table.
  • Update Book
    • Leads to brief table, brief table leads to "add book" format table with details.
  • Issue Book
    • Leads to form, redirects to issued books
  • Issued Books
    • Displays Table with Book details, no borrower details.
  • Returned Books
    • We make use of where="Returned". Display Name, Surname, Book title, return date and comment.
  • Outstanding Books
    • We display all books outstanding; Book Title, Return Date, Name and surname with option to view borrower details
